4. Membuat Link

Hyperlink atau link dibedakan menjadi 2, yakni:

  1. Internal Link
  2. Eksternal Link/ untuk Backlink :V

A. Internal Link

Internal link bisa digunakan tanpa harus menggunakan protokol HTTP, misalnya href="halaman-2.html".

Nah untuk ke halaman lain yang berbeda folder disa menggunakan href="../" untuk naik ke atas folder dan tambahkan nama folder untuk turun nama folder misalnya href="/folder/halaman-3"

Contoh:

  • Click di sini untuk pindah ke halaman 2.

  • Click di sini untuk pindah ke halaman 3.

B. Eksternal Link

Eksternal link harus dibuat dengan menggunakan protokol HTTP, misalnya href="http://google.com".
